# Mohd Nadeem Siraj - Personal Portfolio (Version 4)

A premium, modern, and fully responsive personal portfolio website designed for an IT Infrastructure & Desktop Support professional.

## File Structure

```text
mohd_nadeem_portfolio_v4/
├── index.html        # Main HTML file (updated with Portfolio.png and swapped testimonial names)
├── README.md         
└── assets/
    ├── css/
    │   └── style.css # Styling and responsive media queries
    ├── js/
    │   └── script.js # Logic for animations, form, toggles, and cursor
    └── images/       # Directory for visual assets. The user is recommended to place Portfolio.png here.
```

## How to Run Locally

1. Download or clone the repository.
2. Ensure you place the actual photo file named **`Portfolio.png`** into the `assets/images/` directory.
3. Open `index.html` in any modern web browser.
